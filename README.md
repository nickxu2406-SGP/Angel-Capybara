# 🌿 卡皮巴拉和洋洋 🌿

一个可爱的卡皮巴拉和洋洋主题小游戏！

## 🎮 游戏介绍

**第一关：洋洋喂卡皮巴拉吃水果**

- 点击水果喂给卡皮巴拉
- 每个水果 +5 分
- 需要在 10 秒内点击水果，超时扣 5 分
- 喂完 20 个水果即可过关！

## 🚀 在线游玩

👉 [点击这里开始游戏](https://nickxu2406-sgp.github.io/Angel-Capybara/)

## 🛠 技术栈

- React + TypeScript
- Vite
- Tailwind CSS

## 📦 本地运行

```bash
npm install
npm run dev
```
